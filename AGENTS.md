# AGENTS.md

Model-agnostic instructions for contributors (human or LLM).

## Defaults

- Follow repository conventions and use `pnpm` for scripts.
- Prefer existing utilities/types over inline ad-hoc definitions.
- Do not modify the two-phase EIP-712 registration flow without explicit security review.

## Task Management

- If the Archon MCP server is available, follow the Claude/Archon workflow in `claude.md`.
- If Archon is unavailable, proceed with local planning or ask the user for guidance.
